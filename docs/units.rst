
Units and notation
========================================

The methods in MHKiT assume data is stored in SI units, for example:

* Acceleration = :math:`m/s^2`
* Current = :math:`A`
* Distance = :math:`m`
* Energy = :math:`J`
* Frequency = :math:`Hz` 
* Mass = :math:`kg`
* Power = :math:`W`
* Pressure = :math:`Pa`
* Time = :math:`s`
* Velocity = :math:`m/s`
* Voltage = :math:`V`
* Volume = :math:`m^3`

The following notation is used in MHKiT-Python

* :math:`f` = Frequency
* :math:`Tp` = Peak period
* :math:`S` = Spectral density
